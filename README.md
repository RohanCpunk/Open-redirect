# Open-redirect
Searching parameters to look for open redirect 

PARAMETERS
- ?url+http://{target}
- ?url+https://{target}
- ?next=http://{target}
- ?next=https://{target}
- ?url=https://{target}
- ?url=http://{target}
- ?url=//{target}
- ?url=$2f%2f{target}
- ?next=//{target}
- ?next=$2f%2f{target}
